# LashedByLiyah | Senior Capstone Project
**Project Omnia | Technical Implementation**

## Overview
LashedByLiyah is a full-stack e-commerce and booking solution designed for a campus-based boutique. The application integrates real-time scheduling with a dynamic inventory system.

## Technical Stack
- **Frontend:** React.js, React Router
- **Backend:** Supabase (PostgreSQL)
- **Authentication:** Supabase Auth
- **Infrastructure:** Google Cloud Platform (GCP)

## Key Features
- **Dynamic Inventory:** Real-time filtering for Services, Training, and Apparel.
- **Booking Engine:** Integrated availability calendar with double-booking prevention.
- **Login Handshake:** Merges guest localStorage carts with user accounts upon authentication.
- **Admin Dashboard:** Full CRUD control for orders, inventory, and schedule management.
